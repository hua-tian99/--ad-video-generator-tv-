# Professional TV Ad Prompt Generator

> Claude Code skill for generating video prompts for Seendance 2.0 AI video generation model. Creates professional TV-style video (60+ seconds) prompts with 4K quality and cinematic production values, primarily for healthcare and medical products.

**Key Features:**
- Claude Code skill with interactive Q&A workflow
- Generates prompts for Seendance 2.0 video model via 即梦 (Jimeng) platform
- 16:9 horizontal format for TV and professional media
- 3-layer hook mechanism for viewer engagement
- Primarily for healthcare and medical products

# 电视广告视频脚本生成器

## 项目简介

这是一个 **Claude Code 技能**，主要用于保健品和医疗产品领域，用于生成专业电视级广告视频提示词。生成的提示词可直接在 **即梦平台** 使用 **Seendance 2.0 视频生成模型** 生成实际视频。

通过5轮交互式问答收集产品信息，自动生成高质量的4K视频提示词，无需手动剪辑视频。

## 技术栈

- **Claude Code 技能系统**：交互式问答和提示词生成
- **Seendance 2.0**：AI视频生成模型
- **即梦平台**：视频生成服务平台

## 核心特性

### 🎬 专业电视制作标准
- **横屏16:9格式**：4K高画质（3840x2160），适配电视和专业媒体平台
- **专业摄影**：摄影棚拍摄、专业打光、演员表演
- **流畅过渡**：专业转场效果，画面衔接流畅自然

### ⚡ 三层钩子机制（0-3秒）
- **视觉钩子**（0-1秒）：产品特写、专业展示、吸引眼球的视觉元素
- **痛点钩子**（1-2秒）：引发目标用户情感共鸣的痛点描述
- **解决方案钩子**（2-3秒）：立即给出产品定位，建立"问题-解决方案"连接

### 🎯 持续视觉元素
- 产品宣传语贯穿全程（顶部/底部固定位置）
- 品牌logo持续显示
- 促销信息强化记忆

### 📖 完整叙事结构
钩子吸引（0-3秒）→ 痛点引入 → 解决方案展示 → 功能特点 → 案例场景 → 技术背书 → 产品细节 → 行动号召

### 🎬 智能脚本生成
- 交互式5轮问答收集产品信息
- 自动生成2-5个15秒视频片段（总时长30-75秒）
- 场景具体化，避免泛泛描述
- 专业视觉风格统一

## 使用前提

本项目是一个 **Claude Code 技能**，需要以下环境：

- **Claude Code CLI**：Anthropic 官方命令行工具
- **即梦平台账号**：用于使用 Seendance 2.0 模型生成视频
- **使用方式**：在 Claude Code 中加载本技能（通过 skill.md 配置文件）

## 快速开始

### 1. 在 Claude Code 中加载技能

在 Claude Code CLI 中加载本技能（技能会自动读取 skill.md 配置）。

### 2. 启动交互式问答

AI助手将引导你完成5轮问答：

**第1轮：基本信息**
- 产品名称、类别、品牌

**第2轮：钩子设计要素**
- 核心痛点（用于痛点钩子）
- 产品定位（用于解决方案钩子）
- 视觉特点（用于视觉钩子）

**第3轮：功能与场景**
- 核心功能（3-5个）
- 技术特点
- 具体使用场景（必须具体）
- 产品优势

**第4轮：营销与持续视觉元素**
- 促销活动
- 持续宣传语
- 宣传语显示位置
- 价格定位

**第5轮：视觉素材**
- 产品外观特征
- 产品包装特征
- 使用场景图片描述

### 3. 获取生成的视频提示词

Claude Code 将生成包含以下内容的完整提示词：
- 产品信息摘要
- 视频整体规划
- 每个片段的详细提示词（画面、镜头、光影、文字、持续视觉元素等）
- 整体风格说明
- 使用建议

### 4. 在即梦平台生成视频

将生成的提示词复制到 **即梦平台**，选择 **Seendance 2.0 模型**，即可自动生成实际视频，无需手动剪辑。

- [X-Y秒]："[文案内容]"（位置：[屏幕下方/中央/上方]）

- 内容："[宣传语]"

## 与短视频版的区别

| 特性 | 电视版（本项目） | 短视频版 |
|------|------------------|----------|
| **格式** | 横屏16:9（4K） | 竖屏9:16（1080x1920） |
| **平台** | 电视、专业媒体 | 微信视频号 |
| **风格** | 专业摄影、演员、摄影棚 | 手机拍摄感、素人感、真实场景 |
| **剪辑** | 流畅过渡、专业转场 | 快节奏、跳切、每镜头2-5秒 |
| **字幕** | 标准字幕 | 超大字幕、支持无声观看 |
| **片段** | 2-5个15秒片段 | 1-3个15秒片段 |
| **目标** | 深度展示、品牌建设 | 快速种草、社交互动 |

## 模板系统说明

项目包含完整的模板库（`templates/`目录），涵盖：

- **钩子设计模板**：视觉钩子、痛点钩子、解决方案钩子的设计范式
- **场景设计模板**：具体化场景描述的方法和案例
- **持续视觉元素模板**：宣传语、logo、促销信息的展示方式
- **片段结构模板**：不同类型片段的标准结构
- **过渡设计模板**：专业转场效果设计
- **视觉风格指南**：专业摄影、打光、镜头运动的实现方法
