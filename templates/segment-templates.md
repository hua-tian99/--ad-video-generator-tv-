# 15秒片段模板

## 标准3片段结构（45秒总长）

### 片段1（0-15秒）：钩子+痛点+信任建立

**时间分配**：
- 0-1秒：产品特写（视觉钩子）
- 1-2秒：痛点文案出现（痛点钩子）
- 2-3秒：解决方案文案（解决方案钩子）
- 3-6秒：产品展示动作（手部调整）
- 6-15秒：真人出镜讲解（建立信任）

**核心目标**：
- 快速抓住用户注意力（前3秒）
- 建立问题-解决方案连接
- 通过真人出镜建立信任感

**提示词模板**：

```
### 片段1（0-15秒）

**钩子设计**：
- 视觉钩子（0-1秒）：[产品特写描述]
- 痛点钩子（1-2秒）：文案"[痛点描述]"
- 解决方案钩子（2-3秒）：文案"[解决方案]"

**画面描述**：
画面起始于[产品特写]，柔和的光线从[方向]照射，突出产品的[特点]。
随后一只手[动作描述]，展示产品的[功能/特点]。
镜头缓慢[运动方式]，在[X]秒时切换到真人中景，
[性别]身着[服装]，面带[表情]，[手势动作]，
背景是[环境描述]，光影[效果描述]。

**镜头运动**：稳定为主，轻微拉近，切换流畅

**光影效果**：柔和自然，均匀分布，突出产品和人物

**文字内容**：
- 1-2秒："[痛点文案]"（位置：屏幕下方1/3处）
- 2-3秒："[解决方案文案]"（位置：屏幕下方1/3处）
- 6-15秒：真人讲解配合字幕（如需要）

**持续视觉元素**：
- 位置：[顶部/底部]
- 内容："[产品宣传语]"
- 显示时长：贯穿整个片段（0-15秒）

**风格标签**：温馨友好，专业可信，4K高画质

**过渡提示**：片段结尾真人[手势动作]，为下一片段的功能展示做铺垫
```

---

### 片段2（15-30秒）：功能展示+案例场景+技术背书

**时间分配**：
- 15-20秒：真人继续讲解核心功能
- 20-23秒：案例场景展示（文案+画面）
- 23-27秒：科技元素特写（芯片/技术图示）
- 27-30秒：功能文案强化

**核心目标**：
- 展示产品的核心功能和优势
- 通过具体场景增强用户代入感
- 通过技术元素建立专业背书

**提示词模板**：

```
### 片段2（15-30秒）

**画面描述**：
画面延续片段1，真人继续讲解，[手势动作]配合讲解内容。
在[X]秒时，画面切换或叠加显示案例场景文案：
"[场景1]"、"[场景2]"、"[场景3]"，
每个场景停留[X]秒，可配合相应的场景画面或图标。
随后镜头切换到[科技元素描述]，
[芯片/电路板/技术图示]在[背景]前[运动方式]，
光效[描述]，突出产品的技术含量。

**镜头运动**：从稳定到动态，科技元素部分可旋转或光环移动

**光影效果**：科技部分增加光效，如光环、粒子、反射等

**文字内容**：
- 15-20秒："[功能承诺文案]"（如"日常全场景适配"）
- 20-23秒："[场景1]"、"[场景2]"、"[场景3]"（快速切换）
- 23-27秒："[技术特点文案]"（如"新一代智能芯片"）
- 27-30秒："[功能强化文案]"（如"杂音一键过滤"）

**案例场景**：
- [场景1]："[具体场景描述]"
- [场景2]："[具体场景描述]"
- [场景3]："[具体场景描述]"

**持续视觉元素**：
- 位置：[顶部/底部]
- 内容："[产品宣传语]"
- 显示时长：贯穿整个片段（15-30秒）

**风格标签**：专业科技，动感流畅，4K高画质

**过渡提示**：科技光效逐渐淡出，为产品细节展示做准备
```

---

### 片段3（30-45秒）：细节+背书+行动号召

**时间分配**：
- 30-35秒：产品细节特写（小巧、舒适等）
- 35-40秒：专业人士背书（白大褂/专家）
- 40-45秒：产品包装展示+促销信息

**核心目标**：
- 展示产品的细节和品质
- 通过专业人士增强可信度
- 促使用户采取行动（购买/咨询）

**提示词模板**：

```
### 片段3（30-45秒）

**画面描述**：
画面聚焦产品细节，镜头[运动方式]展示产品的[特点1]、[特点2]。
在[X]秒时，切换到专业人士出镜，
[性别]身着[白大褂/专业服装]，面带[表情]，
[手势动作]，展示或讲解产品。
最后镜头切换到产品包装，[开箱动作/展示动作]，
展示完整的产品和配件，背景[描述]。

**镜头运动**：从特写到中景，最后聚焦产品包装

**光影效果**：柔和均匀，突出产品质感和专业氛围

**文字内容**：
- 30-35秒："[产品特点文案]"（如"小巧隐形 久戴无负担"）
- 35-40秒："[适用人群文案]"（如"男女老少全适配"）
- 40-45秒："[行动号召文案]"（如"XX活动福利大放送 重点来了 闭眼冲不踩坑"）

**持续视觉元素**：
- 位置：[顶部/底部]
- 内容："[产品宣传语]"
- 显示时长：贯穿整个片段（30-45秒）
- 最后几秒可放大或强化促销信息

**风格标签**：专业可信，品质感强，4K高画质

**过渡提示**：产品包装打开的动作，展示最终效果，或定格在专业人士的微笑+产品特写
```

## 2片段结构（30秒总长）

适用于功能简单的产品。

### 片段1（0-15秒）：钩子+痛点+解决方案
- 合并钩子设计和产品展示
- 快速建立信任

### 片段2（15-30秒）：功能+场景+行动号召
- 展示核心功能
- 1-2个关键场景
- 促销信息和行动号召

---

## 4-5片段结构（60-75秒总长）

适用于功能复杂、需要详细展示的产品。

### 片段1（0-15秒）：钩子+痛点+信任建立
（同标准结构）

### 片段2（15-30秒）：核心功能展示
- 重点展示2-3个核心功能
- 真人讲解或产品演示

### 片段3（30-45秒）：案例场景+技术背书
- 3-5个具体使用场景
- 技术元素和专业认证

### 片段4（45-60秒）：产品细节+专业背书
- 产品细节特写
- 专业人士出镜

### 片段5（60-75秒）：包装展示+行动号召
- 产品包装和配件
- 促销信息和购买引导
