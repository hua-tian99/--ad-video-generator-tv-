# 持续视觉元素模板

## 什么是持续视觉元素？

持续视觉元素是指在整个视频或多个片段中持续显示的品牌和营销信息，用于：
- 强化品牌记忆
- 突出促销信息
- 增加品牌曝光时长
- 保持视觉连贯性

---

## 核心元素类型

### 1. 产品宣传语/促销信息

**示例**：
- "三月三爱耳月 自然之声福利大放送"
- "XX品牌 专业XX年"
- "新品上市 限时优惠"

**设计要点**：
- 位置：顶部或底部固定位置
- 字体：清晰但不抢眼
- 颜色：与整体色调协调（通常白色或品牌色）
- 大小：适中，不干扰主要内容
- 显示时长：贯穿整个视频或特定片段

### 2. 品牌Logo

**设计要点**：
- 位置：角落固定位置（通常右上角或左上角）
- 大小：小巧，不干扰主要内容
- 透明度：可适当降低透明度（80-90%）
- 显示时长：贯穿整个视频

### 3. 产品名称标识

**设计要点**：
- 位置：与宣传语配合，或单独显示
- 字体：品牌字体，清晰易读
- 显示方式：持续显示或间歇出现

## 位置布局方案

### 方案A：顶部宣传语 + 角落Logo

```
┌─────────────────────────────────────┐
│ [Logo]    产品宣传语/促销信息        │
│                                     │
│                                     │
│         主要内容区域                 │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

### 方案B：底部宣传语 + 角落Logo

```
┌─────────────────────────────────────┐
│                            [Logo]   │
│                                     │
│         主要内容区域                 │
│                                     │
│                                     │
│    产品宣传语/促销信息               │
└─────────────────────────────────────┘
```

### 方案C：顶部+底部双重信息

```
┌─────────────────────────────────────┐
│ [Logo]    品牌名称                   │
│                                     │
│         主要内容区域                 │
│                                     │
│                                     │
│    促销信息/行动号召                 │
└─────────────────────────────────────┘
```

## 设计原则

### 1. 不干扰主要内容
- 持续视觉元素应该是辅助性的，不能抢夺主要内容的注意力
- 字体大小适中，不要过大
- 颜色选择要与背景协调

### 2. 保持一致性
- 在所有片段中保持相同的位置和样式
- 字体、颜色、大小保持统一
- 增强品牌识别度

### 3. 与整体风格协调
- 颜色与视频整体色调协调
- 字体与品牌风格一致
- 不破坏视频的美感

### 4. 适时强化
- 在关键时刻可以放大或高亮显示
- 在行动号召阶段可以增强促销信息
- 在片段结尾可以强化品牌名称

---

## 不同产品类型的推荐方案

### 保健品类
- 顶部：品牌名称 + "专业XX年"
- 底部：促销信息（如有）
- Logo：右上角

### 医疗器械类
- 顶部：品牌名称 + 专业认证信息
- 底部：产品型号或系列名称
- Logo：左上角

### 健康用品类
- 顶部：促销活动信息
- 底部：品牌slogan
- Logo：右上角

---

## 示例

### 示例1：助听器产品

**持续视觉元素**：
- 顶部："三月三爱耳月 自然之声福利大放送"
- 位置：画面顶部居中
- 字体：白色，清晰易读
- 大小：占画面宽度的60-70%
- 显示时长：贯穿所有片段（0-35秒）

### 示例2：按摩仪产品

**持续视觉元素**：
- 顶部："XX品牌 专业理疗20年"
- 底部："新品上市 限时8折"
- Logo：右上角，半透明
- 显示时长：顶部贯穿全程，底部在最后15秒强化

## 检查清单

设计持续视觉元素后，检查以下要点：

- [ ] 位置是否固定且不干扰主要内容？
- [ ] 字体是否清晰易读？
- [ ] 颜色是否与整体色调协调？
- [ ] 大小是否适中？
- [ ] 是否在所有片段中保持一致？
- [ ] 是否有效强化品牌记忆？
- [ ] 促销信息是否清晰传达？
