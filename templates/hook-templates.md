# 钩子设计模板

## 三层钩子机制（0-3秒）

### 第1层：视觉钩子（0-1秒）

**目标**：用强烈的视觉元素立即抓住用户注意力

**模板选项**：

#### 模板A：产品特写手持展示
```
画面从模糊逐渐清晰，一只手优雅地托举着[产品名称]，产品位于画面中央，
柔和的光线从侧面照射，突出产品的[视觉特点：小巧/科技感/专业感]。
镜头缓慢拉近，聚焦产品细节。
```

#### 模板B：产品动态展示
```
[产品名称]在纯色背景前缓慢旋转，光线随着旋转角度变化，
产品表面反射出柔和的光泽，突出其[材质特点]和[设计美感]。
```

#### 模板C：产品使用场景特写
```
画面聚焦在[使用场景]中的产品，[人物动作]正在[使用产品]，
镜头从侧面特写产品与[身体部位]的接触，展现其[舒适性/便捷性]。
```

---

### 第2层：痛点钩子（1-2秒）

**目标**：引发目标用户的情感共鸣，让用户意识到"这说的就是我"

**文案公式**：
```
[目标人群] + [痛点描述] + [情感词]
```

**示例**：
- "家人们听不清太闹心" （助听器）
- "腰酸背痛太难受了" （按摩仪）
- "睡眠质量差很困扰" （助眠产品）
- "血压不稳心里慌" （血压计）
- "关节疼痛走不动" （护膝产品）

**展示方式**：
- 位置：画面下方1/3处
- 字体：清晰易读，白色或黄色
- 动画：从下方滑入或淡入
- 时长：1-2秒

---

### 第3层：解决方案钩子（2-3秒）

**目标**：立即给出产品定位，建立"问题-解决方案"的心理连接

**文案公式**：
```
[行动词] + [产品名称] + [效果承诺]
```

**示例**：
- "别愁 [品牌名][产品名]封神" （强势定位）
- "[产品名]一招搞定" （简单直接）
- "[产品名]让你告别[痛点]" （解决方案）
- "有了[产品名] [痛点]不再是问题" （承诺式）

**展示方式**：
- 位置：画面下方1/3处（接续痛点文案）
- 字体：比痛点文案稍大，强调品牌名
- 动画：痛点文案淡出后立即出现
- 时长：1-2秒

---

## 完整钩子设计示例

### 示例1：助听器产品

**视觉钩子（0-1秒）**：
```
画面从模糊逐渐清晰，一只手优雅地托举着小巧的助听器，
产品位于画面中央，柔和的光线从侧面照射，突出其精致小巧的特点。
镜头缓慢拉近，聚焦产品的流线型设计。
```

**痛点钩子（1-2秒）**：
```
文案："家人们听不清太闹心"
位置：画面下方，白色字体，从下方滑入
```

**解决方案钩子（2-3秒）**：
```
文案："别愁 自然之声助听器封神"
位置：画面下方，品牌名加粗，淡入效果
```

---

### 示例2：按摩仪产品

**视觉钩子（0-1秒）**：
```
按摩仪在纯白背景前缓慢旋转，蓝色LED灯光闪烁，
产品表面反射出科技感的光泽，突出其现代化设计。
```

**痛点钩子（1-2秒）**：
```
文案："腰酸背痛太难受了"
位置：画面下方，黄色字体，从左侧滑入
```

**解决方案钩子（2-3秒）**：
```
文案："XX按摩仪一招搞定"
位置：画面下方，品牌名放大，淡入效果
```

---

## 钩子设计检查清单

设计完钩子后，检查以下要点：

- [ ] 视觉钩子是否在1秒内抓住眼球？
- [ ] 痛点钩子是否精准描述目标用户的痛点？
- [ ] 痛点描述是否引发情感共鸣？
- [ ] 解决方案钩子是否立即给出产品定位？
- [ ] 三层钩子是否在3秒内完成？
- [ ] 文案是否简洁有力（每句不超过10个字）？
- [ ] 视觉和文案是否协调配合？
