# 广告视频提示词生成器

## 描述

专门针对保健品和医疗产品的广告视频生成工具。基于成功的市场案例提取核心模式，通过交互式问答收集产品信息，生成专业的15秒视频片段提示词，包含完整的钩子设计、场景展示和过渡方案。

## 适用场景

- 保健品广告视频制作
- 医疗器械宣传视频
- 健康产品营销视频
- 需要将产品信息转化为视频脚本
- 无现成案例视频，仅有产品信息和图片

## 核心功能

1. **钩子设计器**：前3秒的视觉+痛点+解决方案三层钩子
2. **场景具体化**：将泛泛的"日常使用"转化为具体生活场景
3. **持续视觉元素**：贯穿全程的品牌强化设计
4. **15秒片段拆分**：智能拆分为2-5个片段，适配视频生成工具
5. **专业过渡设计**：确保片段间流畅衔接
6. **完整提示词生成**：包含画面、镜头、光影、文案的详细描述

## 输入要求

- 产品基本信息（名称、类别、品牌）
- 目标受众痛点
- 产品核心功能和卖点
- 具体使用场景（3-5个）
- 营销信息（促销、价格等）
- 产品图片描述（可选）

## 输出内容

- 完整的视频脚本（markdown格式）
- 每个15秒片段的详细提示词
- 钩子设计方案
- 片段间过渡设计
- 持续视觉元素规划

## 使用方法

调用此skill后，系统将通过交互式问答收集产品信息，然后自动生成专业的视频提示词。

## 基于的成功案例

本skill基于一个35秒助听器广告案例提取的核心模式：
- 三层钩子机制（0-3秒快速抓住用户）
- 具体场景展示（"居家嘴嘘"、"陪娃读绘本"等）
- 持续品牌强化（宣传语贯穿全程）
- 专业叙事结构（钩子-痛点-解决方案-功能-场景-技术-细节-行动）
